# 🔄 Loop Engineering Template

> AI-Agent-Driven Software Development Methodology

**Work → Learn → Improve** — A continuous cycle where AI agents grow more effective with every session by persisting knowledge as reusable **skills**.

## Quick Start

```bash
gh repo create my-project --template shira022/loop-engineering-template --public
git clone https://github.com/your-org/my-project.git
cd my-project
# Launch your AI agent and say: "Bootstrap this project"
```

## Features

- **9 built-in skills** — orchestrating the loop lifecycle
- **agentskills.io compatible** — works with Hermes, Opencode, Claude Code, Codex, Gemini CLI, Cursor, GitHub Copilot
- **CI/CD** — 5 GitHub Actions workflows including CodeQL, Dependabot, and Release
- **Git Flow** — `main` / `develop` / `feature/*` / `release/*` / `hotfix/*`
- **Sub-agent system** — explorer, implementer, verifier with provider-agnostic YAML
- **Automation-ready** — scheduled CI triage, configurable schedules
- **Language-agnostic** — Python, TypeScript, Rust, Go, Java, Kotlin, Swift, C#, and more
