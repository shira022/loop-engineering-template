# 🛠️ Built-in Skills

| Skill | Description | Trigger |
|-------|-------------|---------|
| **loop-engineer** | Session orchestrator | Every session start |
| **knowledge-harvest** | Extracts structured learnings | After 5+ tool calls |
| **skill-crafter** | Creates new skills when patterns repeat 3x | On pattern threshold |
| **decision-recorder** | Writes ADRs for architectural decisions | On significant decisions |
| **session-reviewer** | End-of-session retrospectives | Session end |
| **triage** | Scheduled CI triage and automation dispatch | Daily or manual |
| **project-bootstrapper** | Scaffolds new projects | First session only |
| **project-manager** | Multi-project task management | On task dispatch |
| **test-policy** | Enforces 80%+ test coverage | Every PR / commit |
